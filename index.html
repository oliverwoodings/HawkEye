DataLog by oliverw92

Datalog is a lightweight and simple logging and rollback plugin. All data is stored in a super-efficient and well designed MySQL schema, which can then be searched with easy-to-use commands. It comes with a PHP/JQuery-based web app for browsing the data that has been logged. The web app has numerous tools such as filtering, block selection etc, allowing you to quickly and easily check up on griefers/thieves on your server.
